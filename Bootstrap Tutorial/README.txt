in questo file ho scritto dei dati personali per personalizzarlo il più possibile, oltre a cambiare i vari colori e le immagini ho aggiuto un testo inerente alla propria posizione.
Bootstrap è un framework di sviluppo open-source che chiunque può utilizzare gratuitamente. 
Ti consente di creare rapidamente i progetti e creare pagine Web.
Con Bootstrap puoi creare pagine Web complesse da HTML standard e personalizzarle in base alle tue esigenze. 
Bootstrap include anche numerosi plug-in jQuery in grado di fornire funzionalità aggiuntive come pulsanti, descrizioni comandi e altro.
Essos ti offre molte scorciatoie per la creazione di pagine Web che ti faranno risparmiare tempo ed energia. 
Tutto ciò di cui hai bisogno è una conoscenza di base di HTML e CSS per creare risposte reattive, mobili e compatibili con tutti i browser moderni.
